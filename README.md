# 🧮Calculator

A modern and interactive calculator built using **HTML, CSS, and JavaScript**.  
This project performs basic arithmetic operations and includes keyboard support, sound effects, and a clean user interface.

---

## 🚀 Features

- Basic arithmetic operations (+, −, ×, ÷)
- Keyboard input support
- Button click sound effect
- Calculation history
- Clear (C) and Delete (DEL) functionality
- Responsive and modern UI
- Smooth button animations

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## 🎮 How to Use

1. Enter numbers using buttons or keyboard.
2. Use operators (+ − × ÷) for calculations.
3. Press **=** or **Enter** to calculate.
4. Use **C** to clear the display.
5. Use **DEL** or **Backspace** to delete the last character.
